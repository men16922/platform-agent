# Engineering — Agentic Operations (Bible → Interpretation)

This directory defines the **AI Agent Operations Harness** for this repository across 6 core concepts. The structure follows **Bible → Interpretation**:
- **Bible** (`*_ENGINEERING.md`) = **General/portable** conceptual document. Not bound to any specific repository (can be reused in other projects).
- **Interpretation** (`interp/INTERPRETATION.md`) = Application document mapping these concepts to **actual files, commands, and mechanisms in this repository**.

The Bible defines "What/Why", and the Interpretation defines "How in this repository".
